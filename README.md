## 🚀 Proyectos Principales

### 🌾 CRIBA - Plataforma de Gestión Agrícola
* **Descripción:** Aplicación móvil nativa diseñada para la administración y monitoreo de cultivos agrícolas. Integra mapas interactivos para navegación en terrenos de siembra y un sistema seguro de autenticación de usuarios conectados a bases de datos.
* **Tecnologías:** Android Studio, Google Maps API, Java/Kotlin, MySQL.

### 📚 Auria - Premium Bookstore
* **Descripción:** Sitio web moderno para una librería premium con una interfaz sofisticada inspirada en el diseño Art Deco. Cuenta con layouts responsivos, flujos dinámicos de inicio de sesión y gestión integral de usuarios backend.
* **Tecnologías:** HTML5, CSS3, JavaScript, Node.js, MySQL.

### 🛠️ Ferretería El Zapote
* **Descripción:** Sistema de punto de venta e inventario móvil estructurado para la gestión comercial en tiempo real. Incluye un módulo interactivo de carrito de compras, catálogo de herramientas y control dinámico de stock.
* **Tecnologías:** Android Studio, Java, Arquitectura de Interfaces Móviles.
